# Screan

A simple, elegant screenshot studio for creating App Store screenshots.

## Features

- **Multiple device formats**: iPhone (6.7", 6.5", 5.5"), iPad (12.9", 11"), Mac
- **Position presets**: Top, Center, Bottom layouts
- **Customizable text**: Title and subtitle with color picker
- **Background colors**: Fully customizable
- **Device frame**: Optional border around screenshots
- **Shadow**: Add depth to your screenshots
- **Multi-format export**: Export all formats at once as a ZIP file
- **Drag & drop**: Easy screenshot import
- **Dark/Light mode**: Toggle between themes

## Usage

1. Drag & drop your screenshots or click "Add Screenshots"
2. Select a position preset (Top, Center, Bottom)
3. Customize title, subtitle, and colors
4. Choose which device formats to export
5. Click "Exporter" to download a ZIP with all formats

## Tech Stack

Pure HTML, CSS, and JavaScript. No dependencies except JSZip for export.
